# Editkaro.in — Dark neon 3D theme website

## Setup
1. Put these files in one folder: index.html, portfolio.html, about.html, contact.html, styles.css, scripts.js, img1.png, gaming.png, commerce.png, team images.
2. Open `scripts.js` and replace `REPLACE_WITH_YOUR_WEB_APP_URL` with your deployed Apps Script Web App URL.
3. Create Google Sheet → Extensions → Apps Script → paste google_apps_script.gs → Deploy as Web App (Execute as: Me, Who has access: Anyone).
4. Test forms (email subscription & contact). Rows will appear in Sheets (Emails, Contacts).

## Deploy
- Push to GitHub & enable Pages, or deploy to Netlify/Vercel (drag & drop).

## Notes
- Replace placeholder thumbnails (gaming.png, commerce.png, team1.png etc.) with your assets.
- For production security, restrict Apps Script access and add spam protection.
